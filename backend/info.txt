===== Library =====
1. Fiber = go get github.com/gofiber/fiber/v2 (web framework)

2. GodotEnv = go get github.com/joho/godotenv (utk mengambil data dari .env/environment variable)

3. MySQL = go get -u github.com/go-sql-driver/mysql (driver MySQL)

4. Validator = go get github.com/go-playground/validator/v10 (Validasi)

5. JWT = go get github.com/golang-jwt/jwt/v5 github.com/gofiber/contrib/jwt (Authentication & Authorization)

6. Bcrypt = go get golang.org/x/crypto/bcrypt (Hashing Password)

7. CORS = go get github.com/gofiber/fiber/v2/middleware/cors (CORS)

8. Testify = go get github.com/stretchr/testify (Unit Test)
